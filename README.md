# WestJet Access Control Wallpaper

Fondos de pantalla para **Samsung Galaxy** (A16, A15, A14, etc.) con posiciones de control de acceso WestJet.

## Dispositivos soportados

| Modelo | Resolución |
|--------|------------|
| Galaxy A16 / A15 / A25 | 1080 × 2340 |
| Galaxy A14 / A13 | 1080 × 2408 |
| Galaxy A35 / A54 | 1080 × 2340 |
| Galaxy A06 / A05 | 720 × 1600 |

## Posiciones

RAMPA 1, RAMPA 2, OBSERVER RAMPA, ADUANA, etc.

## Uso

```bash
npm install
npm run dev
```

Abre la app en el navegador del Samsung, elige posición y descarga el PNG. Luego aplícalo como fondo de pantalla en **Ajustes → Pantalla → Fondo de pantalla**.
